# 常见问题解答 (FAQ)

### 关于本软件

**Q: 本软件与 Bilibili (B站) 官方有什么关系？**  
A: **没有任何关系。**
本软件（Utopia Music）是由第三方开发者基于兴趣开发的非官方客户端，与上海幻电信息科技有限公司（Bilibili）无任何隶属、赞助或授权关系。

**Q: 项目会一直维护更新吗？**  
A: **不一定。**
这是一个开源项目，更新频率取决于开发者的业余时间。此外，本软件依赖 Bilibili 现有的 API 接口。如果 Bilibili 官方在未来采取技术手段全面封禁第三方接口调用，本项目可能会在不可抗力下被迫停止维护。

**Q: 我想捐赠或赞助这个项目，渠道在哪里？**  
A: **我们不接受任何形式的资金捐助。**
如果您希望表达支持，请考虑向正规慈善机构或开源组织（如 OSI）捐款。对于本项目，您在 GitHub 仓库点击 **Star** 或提交代码（Pull Request）就是最好的支持。

### 安全与隐私

**Q: 我的账号信息安全吗？会被记录或盗取吗？**  
A: **本软件是完全开源的，您可以随时审查代码。**
我们承诺：本软件仅将您的请求发送至 Bilibili 官方服务器以获取必要数据。应用本身**绝不会**向任何第三方服务器上传您的账号密码、Cookie 或任何个人隐私信息。

**Q: 使用本软件是否存在 B 站账号被封禁的风险？**  
A: **理论上存在风险，请您自行评估。**
虽然目前尚未收到因正常使用本软件导致封号的反馈，但使用第三方客户端本质上违反了 Bilibili 的服务条款。建议您仅将本软件用于正常的音乐播放，避免进行高频抓取或其他可能被判定为异常流量的操作。

### 存储与数据

**Q: 软件会占用多少存储空间？**  
A: 实际占用空间取决于您的使用设置和下载量，大致构成如下：
* **应用本体**：约 50MB 左右。
* **音频缓存**：默认限制为 200MB（可在设置中调整大小或关闭）。
* **其他缓存**：图片及元数据缓存通常在 10MB 左右。
* **离线下载**：取决于您下载歌曲的数量、长度及音质。

**注意**：如果您发现本软件占用了异常巨大的存储空间（远超上述范围），可能是触发了缓存逻辑 Bug。请尝试在设置中清理缓存，并前往 GitHub 仓库提交 Issue 反馈。
