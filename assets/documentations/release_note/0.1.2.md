## 我应该下载哪个文件？
### 安卓
常见主流的近五年左右生产的手机：UtopiaMusic-x.x.x-Android-v8a.apk
非常老旧的手机：UtopiaMusic-x.x.x-Android-v7a.apk
桌面模拟器或x86架构的chrome book：UtopiaMusic-x.x.x-Android-x86_64.apk
### iPhone
下载：UtopiaMusic-0.1.1-iOS-universal.ipa  
请使用[AltStore](https://altstore.io/)进行侧载安装。具体教程请参考[官方文档](https://faq.altstore.io/)。

### macOS
UtopiaMusic-0.1.1-macOS-universal.dmg

### Windows
推荐使用：
- UtopiaMusic-x.x.x-Windows-x64-setup.exe
- UtopiaMusic-x.x.x-Windows-x64-setup.msi （暂时没有，我还在研究怎么打包）

不希望进行安装，可以使用便携版：  
UtopiaMusic-x.x.x-Windows-x64-portable.zip

### Linux
推荐使用，通用不限平台：
- UtopiaMusic-x.x.x-Linux-x64.AppImage

RHEL、CentOS、Fedora等 (RedHat系)：
- UtopiaMusic-x.x.x-Linux-x64.rpm

Debian、Ubuntu等 (Debian系)：
- UtopiaMusic-x.x.x-Linux-x64.deb
