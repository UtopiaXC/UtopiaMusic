## 注意
如果您正使用0.1.1版本并且是安卓手机，请手动到[GitHub](https://github.com/UtopiaXC/UtopiaMusic/releases)下载v8a.apk，0.1.1版本存在问题，无法自动匹配架构。  
请检查您的当前版本，如果当前您正在使用0.1.1版本，请**卸载重装**到本版本。由于签名流程变化，跨版本更新需要完全卸载并重新安装，影响范围：Android、Windows（便携版不受影响）。

## 更新内容
1. 解决了一个读写文件的死锁问题  
2. 默认抑制全部网络错误请求  
3. 添加了log的记录与下载功能，如遇bug可打开调试模式，复现，并下载log后提起issue  
4. 解决了在禁用缓存的情况下还会生成缓存的问题  
5. 一些重构和优化  

## 已知问题
1. 未登录的情况下，可能被风控，导致无法播放，请尽量登录使用。
2. iOS上，加载播放列表后，可能需要点击一下开始按钮才能开始，原因未知

## 我应该下载哪个文件？
### 安卓
常见主流的近五年左右生产的手机：UtopiaMusic-x.x.x-Android-v8a.apk  
非常老旧的手机：UtopiaMusic-x.x.x-Android-v7a.apk  
桌面模拟器或x86架构的chrome book：UtopiaMusic-x.x.x-Android-x86_64.apk  

### iPhone
下载：UtopiaMusic-x.x.x-iOS-universal.ipa  
请使用[AltStore](https://altstore.io/)进行侧载安装。具体教程请参考[官方文档](https://faq.altstore.io/)。  

### macOS
UtopiaMusic-x.x.x-macOS-universal.dmg  

### Windows
推荐使用：  
- UtopiaMusic-x.x.x-Windows-x64-setup.exe  
- UtopiaMusic-x.x.x-Windows-x64-setup.msi （暂时没有，我还在研究怎么打包）  

不希望进行安装，可以使用便携版：  
UtopiaMusic-x.x.x-Windows-x64-portable.zip  

### Linux
推荐使用，通用不限平台：  
- UtopiaMusic-x.x.x-Linux-x64.AppImage  

RHEL、CentOS、Fedora等 (RedHat系)：  
- UtopiaMusic-x.x.x-Linux-x64.rpm  

Debian、Ubuntu等 (Debian系)：  
- UtopiaMusic-x.x.x-Linux-x64.deb
