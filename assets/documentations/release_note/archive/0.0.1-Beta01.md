### Beta01

这是一个测试版本。    
除非您知道您在做什么，否则请不要下载。  
如果您下载了该版本，请不要提交任何关于该版本的稳定性相关issue，这是一个还在开发的测试版。

### 包含内容
1. 完整的播放器功能
2. Bilibili推荐、排行榜接口、搜索接口、登录的实现
3. 本地歌单与B站收藏夹、收藏合集的实现
4. 其他相关设置

### 已知问题
1. 启动时推荐页刷新较慢
2. 播放列表模式管理比较混乱，快速切换、替换歌单的时候可能存在幽灵曲目等情况
3. iOS端播放稳定性较差（由于在Android测试环境下开发，不排除其他平台也有类似的问题），可能由于未知原因卡住导致无法继续播放
4. 由于版本号管理问题，iOS无法匹配GitHub tag导致检测更新存在问题。后续正式版使用标准版本号命名逻辑。
